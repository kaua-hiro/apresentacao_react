O que é React?

React - Biblioteca JavaScript para UI

  • Definição: Biblioteca JavaScript de código aberto para construir interfaces de usuário
  • Criador: Facebook (Meta) em 2013
  • Foco: Camada de visualização (View) das aplicações
  • Filosofia: Declarativo, componentizado e eficiente
  • Usado por: Instagram, Netflix, Airbnb, WhatsApp Web

Componentes

Blocos de Construção Reutilizáveis

O que são: Funções que retornam JSX

function Welcome(props) { return <h1>Olá, {props.name}!</h1>; }
  • Reutilização em toda aplicação
  • Código organizado e limpo
  • Fácil manutenção e teste

JSX - JavaScript + HTML

Sintaxe Poderosa e Intuitiva

O que é: Extensão que permite HTML dentro do JavaScript

const element = <h1>Hello, {name}!</h1>; React.createElement('h1', null, 'Hello, ', name, '!');

Benefício: Código mais legível e produtivo

Virtual DOM

Performance Inteligente

  • Problema: Manipular DOM real é lento
  • Solução React: Cria representação virtual em memória
  • Processo:
    1. Estado muda → Nova árvore virtual criada
    2. Compara com versão anterior (diff)
    3. Atualiza apenas o que mudou no DOM real
  • Resultado: Aplicações até 10x mais rápidas

State vs Props

Gerenciamento de Dados no React

STATE (Estado Interno)

  • • Dados do próprio componente
  • • Pode ser modificado
  • • Exemplo: contador, formulários

PROPS (Propriedades)

  • • Dados passados pelo componente pai
  • • Imutável dentro do componente
  • • Exemplo: nome do usuário, configurações
